# BTSS

#### Basic Terminal String Styling

#### Text decorations

example -

```js
import { dim } from "dim";
dim("this text will be dimmed");
```
1. reset
1. bright
1. dim
1. underscore
1. blink
1. reverse
1. hidden

#### Colors

these colors can be used as font color or background color
example -

```js
import { red, bgred } from "btss";

// red font color
red("text");

// red background color
bgred("text");
```
1.  black
1.  red
1.  green
1.  yellow
1.  blue
1.  magenta
1.  cyan
1.  white
1.  crimson
